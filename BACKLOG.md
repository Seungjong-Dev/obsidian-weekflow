- [ ] vim status bar — 대상 블록 표시 (2026-04-18)
  - **현재**: 하단 스테이터스 표시줄은 `-- MODE -- HH:MM` 형태로 커서 시간만 표시
  - **제안**: 커서가 블록 위에 있거나 서브모드(`cs`/`ce`/`cd`/`ct`)에서 변경 대상이 정해지면, 블록 정보(content·시간 범위·태그)를 함께 표시
  - 예시:
    - Normal + 블록 위: `-- NORMAL -- 14:30 ▸ "회의 준비" 14:00-14:30 work`
    - `cs` 진입: `-- CS -- start: 14:00 → 14:10 (ghost) ▸ "회의 준비"`
    - `cd` 직전: `-- NORMAL -- 14:30 ▸ "회의 준비" → defer to 2026-04-19`
  - 사용자가 다음 커맨드의 영향 대상을 명확히 인지할 수 있게 — 키보드 전용 경험의 feedback 루프 보강
